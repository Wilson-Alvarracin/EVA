Los Colores del HTML

COLORES HEXADECIMALES

Los más populares son los códigos Hex de color; tres bytes números hexadecimal (es decir, compuesto de seis dígitos), con cada byte, o de un par de caracteres en el código Hex, que representa la intensidad de rojo, verde y azul en el color, respectivamente.

COLORES HEXADECIMALES TRASNPARENTES

La base es la misma pero agregando dos cifras al total de 6 para interpretar la transparencia de la misma

RGB

El significado de RGB son los tres colores primarios, rojo, verde y azul. (Red, Green and Blue). Todo color en una pantalla se puede representar como una mezcla de estos 3 colores, indicando la proporción de cada uno con un número del 0 al 255.

RGBA

El sistema es exactamente igual que RGB pero ahora agregamos una cifra mas en el codigo para brindar opacidad o transparencia al color.

COLORES PREDEFINIDOS

Los nombres de colores predefinidos son aquellos que el lenguaje de css puede interpretar tan solo insertándolo textualmente.
